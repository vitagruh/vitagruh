# Система отслеживания поездов

## Возможности

✅ **Логирование по best practices:**
- Консоль (INFO уровень)
- Файл `logs/all.log` (INFO+, ротация 10MB, 5 бэкапов)
- Файл `logs/error.log` (ERROR+, ротация 5MB, 3 бэкапа)
- Детальный формат с именем модуля, функции и строки

✅ **Отслеживание поездов:**
- Проверка каждые **60 секунд** (настраивается)
- Добавление/удаление поездов
- Получение списка отслеживаемых поездов
- Остановка отслеживания (конкретного или всех)

## Быстрый старт

```python
from train_tracker import TrainTracker
import asyncio

async def main():
    # Создаем трекер (интервал 60 сек по умолчанию)
    tracker = TrainTracker()
    
    # Добавляем поезда
    tracker.add_train("SAPSAN-752A")
    tracker.add_train("LASTOCHKA-802M")
    
    # Получаем список
    trains = tracker.get_tracked_trains()
    print(f"Отслеживаем: {trains}")
    
    # Запускаем мониторинг
    await tracker.start()
    
    # Работаем...
    await asyncio.sleep(300)  # 5 минут
    
    # Удаляем конкретный поезд
    tracker.stop_tracking("SAPSAN-752A")
    
    # Или останавливаем всё
    tracker.stop_tracking()  # Все поезда
    await tracker.stop()     # Остановить цикл

if __name__ == "__main__":
    asyncio.run(main())
```

## API

### Методы управления

| Метод | Описание |
|-------|----------|
| `add_train(train_number)` | Добавить поезд |
| `remove_train(train_number)` | Удалить поезд |
| `get_tracked_trains()` | Получить список всех поездов |
| `stop_tracking(train_number=None)` | Остановить отслеживание (None = все) |
| `start()` | Запустить цикл проверки (async) |
| `stop()` | Остановить цикл проверки (async) |
| `is_running()` | Проверить статус работы |

### Параметры

```python
# Настройка интервала проверки
tracker = TrainTracker(check_interval=60)  # 60 секунд по умолчанию
```

## Логи

Все логи INFO уровня пишутся:
- **В консоль** - краткий формат: `21:19:50 | INFO | Сообщение`
- **В файл** `logs/all.log` - полный формат: `2026-04-06 21:19:50 | INFO | train_bot | add_train:31 | Сообщение`
- **В файл** `logs/error.log` - только ошибки (ERROR+)
